# netspeed-cli

Internet speed test CLI tool with beautiful UI.

## Installation

### Node.js (npm)
```bash
npm install -g netspeed-cli
```

### Python (pip)
```bash
pip install netspeed
```

## Usage

```bash
netspeed
```

## Features

- Beautiful terminal UI with spinners
- Download speed test
- Upload speed estimation
- Ping test
